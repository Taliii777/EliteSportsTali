# Configuración del Widget de Instagram

## Opción 1: Elfsight (Recomendado - Gratuito)

1. Ve a: https://elfsight.com/es/instagram-feed-instashow/
2. Haz clic en "Get Started Free"
3. Crea una cuenta gratuita
4. Configura el widget:
   - Selecciona "Instagram Feed"
   - Ingresa el usuario: `edelitesportsmanagement_`
   - Personaliza el diseño (opcional)
5. Copia el código del widget que te proporcionen
6. Reemplaza el contenido del archivo `app/components/InstagramPublicFeed.jsx` con el código que te den

## Opción 2: SnapWidget (Alternativa)

1. Ve a: https://snapwidget.com/
2. Crea una cuenta gratuita
3. Crea un nuevo widget para Instagram
4. Ingresa el usuario: `edelitesportsmanagement_`
5. Copia el código del iframe
6. Reemplaza el iframe en `app/components/InstagramPublicFeed.jsx`

## Opción 3: Usar el código actual (Temporal)

El código actual usa SnapWidget directamente, pero puede requerir configuración.
Si no funciona, sigue las instrucciones de la Opción 1 o 2.
